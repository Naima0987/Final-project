# SafeNest

## Author
Naima Abdullahi

## Description
SafeNest is a child shelter management and donation website created to support vulnerable children by connecting users with shelters in need of help. The platform allows users to learn about different shelter categories and submit donations through an interactive donation form. The project promotes awareness, community support, and access to basic needs for children living in difficult conditions.

## Features
- Multi-page responsive website
- Donation form with validation
- Dynamic donation display using JavaScript
- Donation counter on homepage
- LocalStorage data persistence
- Interactive hover effects and animations
- Responsive layout for mobile and desktop

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Setup Instructions
Open `index.html` in your browser or run the project using Live Server in VS Code.
